## 설치
```bash
pip3 install ddunigma
```

## 사용법
```python
from ddu64 import Ddu64

ddu = Ddu64() # 객체 선언
print(ddu.encode("안녕하세요").encode("utf-8")) # 인코드
print(ddu.decode(".우땨땨이?땨뜌.이.뜌이?이!.우우땨이?우뜌.우땨뜌이이.뜌.우땨땨!이이야").decode("utf-8")) # 디코드
```
